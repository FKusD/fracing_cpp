## 🔍 Улучшенный зум

### Изменения в CarGame.h:
```cpp
static constexpr float VIEW_W = 24.0f;  // было 32.0f
static constexpr float VIEW_H = 13.5f;  // было 18.0f
```

**Эффект**: Машина увеличена на экране примерно на **33%** 📈

### Опционально - динамический зум:
```cpp
float zoomLevel = 1.0f; // добавить member variable
```

Потом можно добавить управление через колесико мыши:
```cpp
// В handleInput()
if (scroll_delta > 0) zoomLevel *= 1.1f;
if (scroll_delta < 0) zoomLevel /= 1.1f;

// В updateCamera()
const glm::mat4 proj = glm::ortho(
    -VIEW_W / (2 * zoomLevel), VIEW_W / (2 * zoomLevel),
    -VIEW_H / (2 * zoomLevel), VIEW_H / (2 * zoomLevel),
    -1.0f, 1.0f
);
```
